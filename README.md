# Портфолио - Фронтенд-разработчик

Мой первый проект портфолио, созданный с использованием HTML, CSS и JavaScript.

- Чистый и минималистичный дизайн
- Список навыков, загружаемый динамически
- Полностью оптимизировано для GitHub Pages

## Технологии
- HTML5
- CSS3
- JavaScript (Vanilla)
